Project Details

Project Name:

Detecting Malware Websites

Developer:

Aniket Walunj

Objective:

Classify URLs as safe or unsafe to help users avoid harmful websites.

Technologies Used:
  • Programming Language: Python
  • Frameworks:
    • Flask: For building the web application.
    • scikit-learn: For developing the machine learning model.
  • Libraries Used:
    • pandas: For data manipulation and analysis.
    • joblib: For saving and loading Python objects (model persistence).
    • TfidfVectorizer: Converts text data (URLs) into numerical vectors.
    • LabelEncoder: Converts categorical labels into numerical format.
  • Algorithms Used:
    • Random Forest: Implemented using the RandomForestClassifier from scikit-learn.
  • Frontend: HTML, CSS, and Jinja Template Language: For creating index.html and project_details.html.
Key Project Details:
  • Development Date: 23-10-2024
  • Accuracy: 97.11% in detecting malicious URLs.
Project Overview:

This project detects malware or unsafe websites based on input URLs, ensuring safe browsing for users. The Random Forest Algorithm is implemented to analyze URL features and classify websites as safe or potentially harmful.
